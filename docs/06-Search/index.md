# Search
The Navigation SDK supports several search methods including: 
- text search 
- proximity search-
- category-based search 
- customizable preferences like fuzzy matching and distance limits 
- search for custom landmarks or within overlays 

For geocoding, the SDK provides: 
- reverse geocoding, converting geographic coordinates into comprehensive address details
- geocoding, which allows locating specific places based on address components. 

The SDK also offers integration with Wikipedia for location-based content, and auto-suggestions to dynamically generate search results while typing.
